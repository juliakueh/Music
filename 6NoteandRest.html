Whole Note

Each note is given four beats.

The foundation of the note system is the entire note.

Every smaller note consists of a portion of the larger note or rest.

The whole note signifies a duration of 4 beats in music (Teacher at Heart Break Institution, 2020).

✗

Whole Rest

The whole rest works really similar as whole note do.

It symbolizes a duration of four beats of silence (Teacher at Heart Break Institution, 2020).

✗

Dotted Half Note

Dotted half note have 3 beats.

Half note is 2 beats and the dotted means 1 beats (Teacher at Heart Break Institution, 2020).

✗

Dotted Half Rest

Same value goes with dotted half rest.

Dotted half rest = 3 beats of silence (Teacher at Heart Break Institution, 2020)

✗

Half Note

A half note is counted as 2 beats.

2 half notes are equivalent as 1 whole note.

2 half notes = 1 whole note (Teacher at Heart Break Institution, 2020)

✗

Half Rest

Half rest has the same count as half note(2 beats).

Every half rest signifies a pause of 2 beats.

2 half rests = 1 whole rest (Teacher at Heart Break Institution, 2020)

✗

Dotted Quarter Note

A dotted quarter note been assigned 1½ beats.

This cause by a quarter note counts as 1 beats while a dot gets ½ beat.

Because a dotted quarter note cannot be equivalent to a whole note, it needs to be combined with notes of other durations to complete the four beats.

2 dotted quarter note + quarter note = 1 whole rest (Teacher at Heart Break Institution, 2020)

✗

Dotted Quarter Rest

The dotted quarter rest holds the same time value as dotted quarter note.

1½ beats of silence is the measured value for the dotted quarter rest (Teacher at Heart Break Institution, 2020).

✗

Quarter Note

A quarter note = 1 beatof sound

1 half note = 2 quarter notse

1 whole note = 4 quarter note (Teacher at Heart Break Institution, 2020)

✗

Quarter Rest

1 quarter rest represent ¼ of whole rest.

To get 4 beats 4 quarter rest is needed (Teacher at Heart Break Institution, 2020).

✗

Eighth Note

An eighth note holds the value of ½ beat.

1 eighth note = ½ quarter note

8 eighth note would only have the same value as 1 whole rest (Teacher at Heart Break Institution, 2020).

✗

Eighth Rest

1 eighth rest means ½ beat of silence.

Only one whole rest would be equal to 8 eighth rest (Teacher at Heart Break Institution, 2020).

✗

Sixteenth Note

Sixteenth note only own ⅛ beat.

This note has the shortest duration in a beat.

Sound as its name, a sixteenth note would only be equivalent to a whole note when there are 16 sixteenth notes (Teacher at Heart Break Institution, 2020).

✗

Sixteenth Rest

This is the rest which represent ⅛ of silence.

Only 16 sixteenth rest = 1 whole rest (Teacher at Heart Break Institution, 2020).
